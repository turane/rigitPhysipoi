rigidPhysipoi

剛体の物理演算が行われてるっぽいオブジェクトを生成することができるライブラリです。

これらのオブジェクトはユーザーに自由な描画を行ってもらうために void display を実装していません。extendsするかどうにかするかして自由な描画を行ってください。

Edge

用途：壁、床、棒

棒の長さを0で生成しようとすると PhysipoiException が throw されます

コンストラクタの概要
Edge(PVector location, float angle, float length)
          始点の座標と、Edgeの角度と長さを指定してEdgeを生成します
Edge(PVector location, PVector to)
          始点と終点を指定してEdgeを生成します

メソッドの概要
abstract void display(float angle)
          Edgeを描画します。中身は空ですが、必ずOverrideして使ってください
void setAngle(float angle)
          Edgeの角度を変更します
float getAngle(float angle)
          Edgeの角度を取得します
void getLength(float angle)
          Edgeの長さを取得します

Sphere

用途：弾、球、玉

球の重量を0で生成しようとすると PhysipoiException が throw されます

球の反発係数の初期値は0.8です

コンストラクタの概要
Sphere(float x, float y, float mass, float radius)
          座標と重量、半径を指定して生成します

メソッドの概要
abstract void display(float angle)
          Edgeを描画します。中身は空ですが、必ずOverrideして使ってください
void applyForce(PVector force)
          力を加えます
void setBound(float boundness)
          反発係数を設定します。0以下の値を設定した場合PhysipoiException がthrowされます
void setVelocity(float sx, float sy)
          速度を設定します
void setGravity(float sx, float sy)
          重力加速度を変更します。デフォルト値は0.098です。
void update()
          加速度を速度に、速度を座標に足します。
boolean checkEdges(Edge edge)
          Edgeとのあたり判定を処理します。あたっていた場合はtrueを返します。
boolean contains(PVector p)
          その点がSphere内に存在するときtrueを返します。
float getR()
          Sphereの半径を返します。ellipseで描画する際は描画時は2倍して直径にして御利用ください。
PVector getVelocity()
          速度を返します。(アドレス渡しではない。)
